# Dependencies
*python3.x*  
*pip3*  
*Google Chrome*  
*PyQt4*   
>`# apt-get install `
  
*Beautiful Soup*  
>`# pip3 install bs4`
  
*Selenium*  
>`# pip3 install selenium`

*Selenium-requests*
>`# pip3 isntall selenium-requests`

# Install & Run
>`# git clone https://github.com/ympo2oo2/Py3SuprBot`  
>`# cd Py3SuprBot`  
>`# python3 Py3SuprBot.py`
